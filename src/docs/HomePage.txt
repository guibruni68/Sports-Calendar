RESPONSIVE BEHAVIOR
================================================================================

  Desktop (>1024px):
    - Full sidebar (283px)
    - Banner + live games side by side
    - Carousels with horizontal scroll

  Tablet (768–1024px):
    - Sidebar collapses to icon-only (72px)
    - Labels hidden, icons centered
    - Banner and live games stack vertically
    - Live games scroll horizontally

  Mobile (<768px):
    - Sidebar hidden entirely
    - Sections use 16px padding
    - All carousels horizontal scroll
    - Footer stacks vertically
