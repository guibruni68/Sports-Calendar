SEARCH BAR COMPONENT
====================
Figma Node: 1:6694
Component Set: "Search Bar"
Type: COMPONENT_SET

OVERVIEW
--------
Search input field with a magnifying glass icon and placeholder text.
Icon and placeholder brighten on hover or focus. Supports controlled input
for real-time search filtering. Used in the app header for searching
teams, games, and other content.

VARIANT PROPERTIES
------------------
State (3 variants):
  Default  — idle, no interaction
  Hover    — mouse over the component
  Tipping  — input is focused / user is typing

Total variants: 3
